---- Instrucciones de compilación -----
Ejecutar le comando: 
$ make
Si desea borrar los ficheros .o y los ejecutables utilizar:
$ make clean

---- Instrucciones de ejecución -----
Para ejecutar correctamente el programa
1º ejecutar el programa 'viewnetreceive':
$ ./viewnetreceive
2º ejecutar el programa 'viewnetsend':
$ ./viewnetsend

---- Funcionamiento del programa -----
Tenemos dos programas.
viewnetreceive -> encargado de recibir mensajes de un socket y mostrarlos por pantalla
viewnetsend -> encargado de enviar un fichero desde un socket a otro
